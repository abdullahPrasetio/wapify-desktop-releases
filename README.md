# Wapify Desktop Releases 🚀

Selamat datang di repositori resmi distribusi aplikasi **Wapify**. Repositori ini khusus digunakan untuk menampung installer aplikasi Wapify versi terbaru untuk semua platform (Windows, macOS, dan Linux).

Wapify adalah API Orchestration & Collaboration platform yang dirancang untuk performa tinggi dan sinkronisasi real-time.

---

## 📥 Download Terbaru

Silakan unduh installer sesuai dengan Sistem Operasi Anda melalui link di bawah ini:

| Platform | Format File | Petunjuk |
|---|---|---|
| **Windows** | `.exe` (Setup) | Unduh dan jalankan installer. |
| **macOS** | `.dmg` (Apple Silicon/Intel) | Buka dmg dan drag Wapify ke Applications. |
| **Linux** | `.AppImage` / `.deb` | Ubah izin menjadi executable dan jalankan. |

👉 **[Klik di sini untuk melihat semua rilis (RELEASES)](https://github.com/abdullahPrasetio/wapify-desktop-releases/releases/latest)**

## 🛠️ Catatan Instalasi

### Windows
Jika Anda melihat peringatan **"Windows protected your PC"** atau **"SmartScreen"**:
1. Klik **"More info"**.
2. Klik **"Run anyway"**.
*Hal ini wajar karena aplikasi belum menggunakan sertifikat berbayar (EV Code Signing).*

### macOS
Aplikasi ini dioptimalkan untuk arsitektur ARM (M1/M2/M3). Jika muncul peringatan keamanan:
1. Buka **System Settings** > **Privacy & Security**.
2. Scroll ke bawah dan klik **"Open Anyway"** pada bagian Wapify.

---

## 🔄 Pembaruan Otomatis (Auto-Update)
Aplikasi Wapify dilengkapi dengan fitur pembaruan otomatis. Setiap kali Anda membuka aplikasi, Wapify akan mengecek apakah ada versi terbaru di repositori ini. Jika tersedia, pembaruan akan diunduh di background dan Anda akan diminta untuk melakukan restart aplikasi.
